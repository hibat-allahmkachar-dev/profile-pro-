# Profil Utilisateur IRIS

## Description

Page de profil utilisateur réalisée dans le cadre du workshop **HTML Masterclass** organisé par IRIS Junior Création.  
Le projet applique les bonnes pratiques HTML5 : sémantique, accessibilité (ARIA), validation W3C et code propre et commenté.  
Il constitue également un exercice complet de workflow Git/GitHub professionnel.


## Aperçu

> Page structurée en 3 sections principales : **À propos**, **Compétences** et un **formulaire de contact** — le tout dans un design cohérent et responsive.



## Technologies utilisées

| Technologie | Usage |
|---|---|
| HTML5 | Structure sémantique de la page |
| Git / GitHub | Versionnement et workflow professionnel |


## Fonctionnalités

- **En-tête** : avatar (`<figure>` / `<figcaption>`), nom, rôle et navigation interne par ancres
- **Section À propos** : biographie avec balises `<section>` et `<p>`
- **Section Compétences** : liste structurée avec `<ul>` / `<li>`
- **Section Contact** : formulaire complet avec `<fieldset>`, `<legend>`, 5 types d'inputs différents (`text`, `email`, `tel`, `date`, `checkbox`) + `<select>` et `<textarea>`
- **Pied de page** : liens réseaux sociaux (GitHub, LinkedIn, Email) et copyright
- **Accessibilité** : attributs `aria-label`, `aria-labelledby`, `role`, `alt`, hiérarchie H1 → H2 respectée, `lang="fr"`
